## 画面要素

### 検索・絞込フォーム

| 項目 | 形式 | 説明 |
|-----|------|------|
| キーワード | テキスト | イベント名、会場名で検索 |
| カテゴリ | セレクトボックス | すべて/コンサート/スポーツ/演劇/フェス/オンライン |
| エリア | セレクトボックス | すべて/東京/大阪/名古屋/札幌/福岡 |
| 開催日 | 日付範囲 | 開始日～終了日 |

### イベントカード

各イベントカードには以下を表示：

| 要素 | 説明 |
|-----|------|
| サムネイル画像 | イベント画像（200x150px） |
| イベント名 | イベント名（クリックで詳細へ） |
| 開催日時 | YYYY年MM月DD日(曜日) HH:MM ～ |
| 会場 | 会場名 |
| 価格帯 | 最低価格 ～ 最高価格 |
| 販売状況 | [残りN枚] / [販売中] / [売り切れ] |
| [詳細を見る]ボタン | イベント詳細ページへ遷移 |

### 並び順

- 新着順（登録日時の降順）
- 開催日順（開催日時の昇順）
- 人気順（販売枚数の降順）

### ページネーション

- 1ページあたり10件表示
- [前のページ] [1] [2] [3] ... [次のページ]

---

## 動作フロー

### 正常フロー

1. 購入者がトップページまたは直接URLでアクセス
2. 公開中のイベント一覧が表示される（デフォルト: 新着順）
3. 検索・絞込条件を指定して[検索]ボタンをクリック
4. 条件に合致するイベントが表示される
5. イベントカードをクリックまたは[詳細を見る]ボタンでイベント詳細へ遷移

### 検索結果が0件の場合

```
検索結果が見つかりませんでした。

検索条件を変更してお試しください。
・キーワードを変更する
・カテゴリやエリアを「すべて」にする
・開催日の範囲を広げる
```

## レスポンシブ対応

### PC (1024px以上)
- イベントカードは横並び（2列）
- サムネイル画像を大きく表示

### タブレット (768px - 1023px)
- イベントカードは横並び（2列、小さめ）
- 検索フォームは2列

### スマホ (767px以下)
- イベントカードは縦1列
- 検索フォームは縦1列
- サムネイル画像は左側に小さく表示

## Constitution遵守チェック

✅ **データモデル中心**: Eventモデルから公開中のイベントを取得  
✅ **検索機能**: キーワード、カテゴリ、エリア、開催日で絞込  
✅ **ソート機能**: 新着順、開催日順、人気順  
✅ **ページネーション**: 大量のイベントにも対応  
✅ **レスポンシブデザイン**: Bootstrap 5で実装  
✅ **未ログインアクセス可**: 会員登録前でもイベント閲覧可能
